# 介绍

该工具为我们提供了前端项目业务中快速常用的工具方法函数，工具库的代码大部分来自网络和社区，并使用了部分
[outils](https://github.com/proYang/outils) 源码，项目全部使用`typescript`开发。目前工具库中提供了`array`、`blob`、`cookie`、`date`、
`device`、`dom`、`number`、`object`、`random`、`regex`、`string`相关的操作方法和部分工具方法。目前工具库中部分工具函数还不够完善，后续会慢慢完善，
也欢迎大家提出建议优化，或直接提`pr`,有任何问题，可在[issue](https://github.com/zguiyang/quick-utils-js/issues) 中提问。
